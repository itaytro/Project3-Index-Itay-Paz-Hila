# DishDiary - Digital Recipe Book
DishDiary is a responsive web-based landing page designed for a personal recipe management system.
This project serves as the foundation for a larger web application, focusing on semantic HTML5 structure, CSS layout techniques, and responsive design principles.

## GitHub Pages Website
You can the project online here: https://itaytro.github.io/Project3-Index-Itay-Paz-Hila/

## Project Requirements & Features
This project was built according to the following specifications:

* Semantic HTML5: Full use of modern structural tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
* Fully Responsive: Optimized for various screen sizes, including a specialized layout for extra-large screens (1500px+) and mobile devices (under 750px).
* Navigation: A functional navigation bar using href="#" for placeholder links, complete with custom tooltips for "not yet available" features.
* Aesthetic Design: A clean, culinary-themed interface featuring Google Fonts (Playfair Display and Allura), semi-transparent overlays, and high-quality background scaling.
* Modern Layouts: Implemented using Flexbox to ensure proper alignment of the logo, navigation, and action buttons.

## Built With
* HTML5 - Structure and semantic content.
* CSS3 - Custom styling, Flexbox, and Media Queries.
* Google Fonts - Typography integration.
